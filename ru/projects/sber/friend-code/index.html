Тимонов Максим

Продукт

Low-code конструктор сервисов. Используя дизайн-систему и отдельно спроектированные smart-элементы, позволяет быстро разрабатывать уникальный дизайн для сервисов в СберДруге.

Аудитория

Подрядчики и внешние разработчики сервисов в Сбер

Метрики продукта

Скорость вывода сервисов. Вовлеченность пользователей.

Моя роль

Контроль проекта. Участие в планировании бэклога и распределния приоритетов компонентной базы.

Дизайн сервиса

Wysiwyg web конструктор позволяет собирать визуальные элементы и экраны для сервисов СберДруга.

Smart elements и шаблоны

Готовые компоненты на базе дизайн системы и заранее запрограмированной логикой часто используемых кейсов позволяют сократить время разработки. Так же есть система шаблонов для часто используемых интерфейсных решений.

Кроссплатформенность

Решение позволяет собирать как адаптивные веб экраны, так и нативные мобильные экраны для тех же сервисов. Т.к. используется единая дизайн система.

Для продуктов

Использует дизайн систему продукта СберДруг и публикует сервисы в его каталог.

Управление сервисом

Есть все необходимо для управление сервисом разработчиком, менеджером и всей командой.

Совместная работа

Есть возможность обмениваться готовыми решениеями в базе шаблонов. Управлять и редактировать одним проектом одновременно несколькими разработчиками.

Проверка, публикация и версионность

Процесс публикация делится на несколько этапов проверок. Каждая публикация создает отдельную версию. Есть возможность менять опубликованную версию на любую стабильную одобренную версию.

Code & accesses

Подключение к GIT или редактирование кода для управления сервисом.

Программирование логики

Frontend полностью формируется в конструкторе и остается только дополнить логикой и управление сервиса.

Отладка и тестирование

Есть консоль выводящая информация о тестовых серверах и тестовые вкладки для отладки на необходимой версии сервиса.

Точка входа, ключи, справочники

Так же есть готовые ключи, точки входа и используемые справочники для использования в сервисе.
