Работа вместе с разработчиками

При рарзработке сценариев и макетов учитываются все возможные решения, поведение пользователя, условия и окружение. Фиксируется в комбинациях в макетах и табличцах в базе знаний о проекте.

Умею работать с Git (GitHub, GitLab), Atom, sublime text, terminal. Развертываю тестовые стенды через Terminal. Участвую в работе команды от ранних стадиях MVP и MLP.

Быстрое прототипирование

Для быстрого прототипирования и исследований использую

  • Figma – анимированные прототипы для команды и удаленных исследований
  • Principle сложной анимации и таймингов разработчикам
  • webflow - для проверки web и html статичной разработки

Далее подбираю инструменты подходящие под ситуацию, лицензии и рабочий поток задач.

‍

Сложные прототипы

Если дешевле составить сложное тестирование для проверки гипотезы, использую:

  • middleman app (ruby on rail) – web прототипы
  • FramerX (react) – для прототипирование мобильной разработки и анимации
  • Webflow (html, js, css) – полная разработка для лендингов и небольших сайтов

Подбор инструментов и сервисов првоожу под проект. Например для исследований со сложными опросами, heatmap и т.д.
